## 快速启动项目

自动运行启动前操作

* 删除node_modules和package-lock.json
* npm install
* npm run serve

## Install
```
    npm install mxj-start -g
```

## Start
在项目文件夹下运行`mxj-start`

## 配置

选项：
  --version    显示版本号                                                 
  -a, --all   是否删除全部node_modules(默认只删除node_modules下的@business目录)                    
  -h           显示帮助信息                

示例：
  mxj-start
